# django-project
## Run the application

```
> virtualenv env
> env\Scripts\activate
> pip install -r requirements.txt
> python manage.py runserver
```

- on `mysite/github_settings.py` chage the following
```
SOCIAL_AUTH_GITHUB_KEY = 'your key'
SOCIAL_AUTH_GITHUB_SECRET = 'your secret key'
```
